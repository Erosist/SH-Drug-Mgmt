# 手动测试脚本

这个文件夹包含手动测试和工具脚本，用于开发过程中的手动验证和调试。

## 文件说明

### `manual_api_test.ps1`
- **用途**: 手动API测试脚本
- **语言**: PowerShell
- **功能**: 测试库存预警API的各个端点
- **使用方法**: 
  ```powershell
  # 确保后端服务正在运行
  cd vue-project/backend
  python run.py
  
  # 在另一个终端运行测试脚本
  .\scripts\manual_api_test.ps1
  ```

## 与自动化测试的区别

- **自动化测试** (`tests/` 文件夹): 用于CI/CD流水线，使用pytest框架
- **手动测试脚本** (`scripts/` 文件夹): 用于开发调试，手动验证API行为

## 注意事项

1. 这些脚本需要手动运行，不会在CI/CD中自动执行
2. 运行前请确保后端服务已启动
3. 可能需要根据实际的测试数据调整脚本中的参数
